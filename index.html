Next é um projeto que envolve a utilização do Arduino para construção de dispositivos eletrônicos interativos, bem como a utilização de aulas de programação usando HTML e CSS para a programação do site do projeto. Esta abordagem visa tornar a construção de dispositivos eletrônicos mais acessível e divertido para os utilizadores, enquanto fornece uma plataforma interativa para compartilhar os projetos e ideias entre a comunidade de programadores.
